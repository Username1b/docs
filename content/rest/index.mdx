# Fusion Feed REST API

The Fusion Feed REST API provides sports schedule information, detailed play and event data, and access to AV streams of sports games from multiple camera angles.

You can get started with [Getting Started with Fusion Feed](#getting_started) or you can check out the [Fusion Feed API Reference](/tag/Fusion-Feed-REST-API-Reference) for detailed documentation of API responses and queries.
