## Authentication

Fusion Feed uses API Authorization Tokens for authentication and authorization.

API Authorization Tokens must be provided in the HTTP Authorization header.

### Requesting an API Authorization Token
If you do not have an API Authorization Token, you can request a token by emailing your API Authorization Token request to fusionfeed@tempus-ex.com. Provide the following information in your API Authorization Token request:
- your name
- your email address
- your company name
- your job title
- the name and description of your application
